== Description ==

Provides a way to view and edit the settings of all blogs in the network.

Once the data is collected from all the blogs the super admin can choose which options to show in the overview.

Blogs can be selected and their options can be edited. Options can be copied from one blog to others.

Can be used to copy options from one blog to many or to clean up unused options. Also provides a way for super admins to see which plugins are activated on which blogs. 

Available in:

* English
* Swedish

= Tip =

You'll probably want to raise Wordpress' maximum memory limit. It needs at least 256MB to collect data from about 330 blogs... 

== Installation ==

1. Unzip and copy the zip contents (including directory) into the `/wp-content/plugins/` directory
1. Activate the plugin sitewide through the 'Plugins' menu in WordPress.
